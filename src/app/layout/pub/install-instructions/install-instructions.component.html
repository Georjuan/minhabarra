Instruções de instalação Minha Barra é um aplicativo web, não necessita de instalação, para utilizar basta acessar o site minhabarra.com.br, para facilitar, vamos criar um atalho de acesso rápido ao site, siga as instruções abaixo.

Adicione o ícone do site à Tela de Inicio

Com a página do site aberta, toque no ícone de menu, que poderá ser um desses abaixo:

Role o menu e localize a opção:

Adicionar à tela inicial ou Adicionar à Tela de Início ou Adicionar à tela ou Adicionar página a > Tela inicial

*No iPhone, caso não veja a opção "Adicionar à Tela de Início", vocÃª pode adicioná-la. Role até a parte inferior da lista, toque em Editar Ações e toque em (+) "Adicionar à Tela de Início".

Uma tela de confirmação será exibida:

Após confirmado, um ícone de atalho para o site Minha Barra será exibido entre seus aplicativos, é só aproveitar!
